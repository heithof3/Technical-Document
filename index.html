For Loops

JavaScript supports different kinds of loops:

  • for -loop through a block of code a number of times
  • for/in - loops through the values of an iterable object
  • for/of - loops through the properties of an object
  • while - loops through a block of code while a specified condition is true
  • do/while - also loops through a block of code while a specified condition is true

The For Loop

The for loop has the following syntax:

myElement.style.display = "for (statement 1; statement 2; statement 3) { // code block to be executed }";

Statement 1 is executed (one time) before the execution of the code block.

Statement 2 defines the condition for executing the code block.

Statement 3 is executed (every time) after the code block has been executed.

Example

myElement.style.display = "for (i = 0; i 5; i++) { text +="The number is " + i + "
" ; }"

From the example above, you can read:

Statement 1 sets a variable for the loop starts (var i = 0).

Statement 2 defines the condition for the loop to run (i must be less than 5).

Statement 3 increases a value (i++) each time the code block in the loop has been executed.

Statement 1

Normally you will use statement 1 to initialize the variable ues in the loop (i = 0).

This is not always the car, JavaScript doesn't care. Statement 1 is optional.

You can initiate many values in statement 1 (separated by comma):

Example

myElement.style.display = "for (i = 0, len = cars.length, text = ""; i < len; i++) { text +=cars[i] + "
" ; }"
" ; }"

And you can omit statement 1 (like when you values are set before the loop starts):

Example

myElement.style.display = "var i = 2; var len = cars.length; var text = ""; for (; i < len; i++) { text +=cars[i] + "
" ; }"

Statement 2

Often statement 2 is used to evaluate the condition of the initial variable.

This is not always the case, JavaScript doesn't care. Statement 2 is also optional.

If statement 2 returns true, the loop will start over again, if it reurns false, the loop will end.



Statement 3

Often statement 3 incresments the value of the initial variable.

This is not always the case,JavaScript doesn't care, and staement 3 is optional.

Statement 3 can do anything like negative increment (i--), positive (i = i + 15), or anything else.

Statment 3 can also be omitted (like when you increment your values inside the loop):

Example

myElement.style.display = "var i = 0; var len = cars.length; for (; i < len; ) { text +=cars[i] + "
" ; i++; }"

For/of Loop

The JavaScript for/of statement loops through the values of an iterable object. It lets you loop over iterable data structures such as Arrays, Strings, Maps, NodeLists, and more:


Syntax

myElement.style.display = "for (variable of iterable) {
// code block to be executed"
};

variable - For every iteration the value of the next property is assigned to the variable. Variable can be declared with const, let, or var.

iterable - An abject that has iterable properties.

Lopping over an Array


Example

myElement.style.display = "let cars = ["BMW", "Volvo", "Mini"];
let text = " "

for (let x of cars) {
Text += x + " ";
}"

Lopping over an String


Example

myElement.style.display = "let language = "JaveScript";
let text = " ";

for (let x of language) {
text += x + " ";}
}"
While Loop

The while loop loops through a block of code as long as a specified condition is true.


Syntax

myElement.style.display = "while (condition) {
// code block to be excuted
}";

ExampleIn the following example, the code in the loop will run, over and over again, as long as a variable (i) is less than 10:


Example

myElement.style.display = "while (i < 10) {
text += "the number is " + i;
i++;
}"

The Do/While Loop

 The do/while loop is a cariant of the while loop. This loop will excute the code block once, before checking if the condition is true, then it will repeat the loop as long as the condition is true.


Syntax

myElement.style.display = "do {
// code block to be excuted
}
while (condition);";

Example

The example below uses a do/while loop. The loop will always be executed at least once, even if the condition is false, because the code block is executed before the condition is tested:


Syntax

myElement.style.display = "do {
text += "The number is " +i;
i++;
}
while (i < 10);";
Break and Continue

The break statement "jumps out" of a loop.

The continue statment "jumps over" one iteration in the loop.

The Break Statement

You have already seen the break statement used in an earlier chapter of the tutorial. it was used to "jump out" of a switch() statment.

The break statement can also be used to jump out of a loop:


Example

myElement.style.display = "for (i = 0; i < 10: i++) {
if (i === 3) { break; }
text += "The number is " + i + " ";
}";

In the example above, the break statement ends the loop ("breaks" the loop) when the loop counter (i) is 3.


The Continue Statement

the continue staement breaks one iteration (in the loop), if a specified condition occurs, and continues with the next iteration in the loop.

This example skips the value of 3:


Example

myElement.style.display = "for (i = 0; i < 10: i++) {
if (i === 3) { continue; }
text += "The number is " + i + " ";
}";

JaveScript Labels

To label JavaScript statements you precede the statements with a label name and a colon:

label:
statements

The break and the continue statements are the only JavaScript statements that can "jump out of" a code block.

Syntax:

break labelname;

continue labelname;

The continue statement (with or without a label reference) can only be used to skip one loop iteration.

The break statement, without a label reference, can only be used to Jump out of a loop or a switch.

With a label reference, the break statement can be used tojump out of any code block:


Example

myElement.style.display = "var cars = ["BMW", "Volvo", "Saab", "Ford"];
list: {
text += cars[0] = " ";
text += cars[1] = " ";
break list;
text += cars[2] = " ";
text += cars[3] + " ";
}

A code block is a block of code between { and }.
